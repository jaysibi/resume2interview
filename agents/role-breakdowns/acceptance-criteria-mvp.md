# Acceptance Criteria — MVP Features

## 1. Resume Upload and Parsing
- Users can upload resumes in PDF or DOCX format.
- The system parses and extracts name, skills, experience, education, and achievements with ≥90% accuracy.
- Users can review and confirm extracted data before proceeding.
- Errors in parsing are clearly reported to the user.

## 2. Job Description (JD) Upload and Parsing
- Users can upload or paste a job description.
- The system extracts mandatory skills, preferred skills, and keywords from the JD.
- Extracted requirements are displayed for user review.

## 3. Skill and Keyword Extraction
- The system displays a list of skills and keywords from both the resume and JD.
- Skills/keywords are categorized (e.g., technical, domain, leadership).
- Users can see overlaps and gaps between resume and JD.

## 4. Gap Analysis and ATS Match Scoring
- The system generates an ATS match score (0–100) based on keyword overlap, semantic similarity, formatting, and achievements.
- A gap report lists missing keywords, weak sections, and suggestions for improvement.
- Users receive actionable recommendations.

## 5. Achievement/Bullet Rewriting and Enhancement
- Weak resume bullets are identified and rewritten using strong action verbs and quantifiable outcomes.
- Rewritten bullets are factually accurate and concise.
- Users can accept or edit suggested rewrites.

## 6. ATS-safe Resume Formatting and Export
- The system generates a resume in DOCX and PDF formats with ATS-safe formatting (single column, no graphics/tables, standard headings).
- Exported resumes pass ATS validation checks.

## 7. ATS Score and Keyword Report Generation
- Users receive a downloadable report with ATS score, missing keywords, and recruiter-style feedback.
- The report is clear, actionable, and visually clean.
