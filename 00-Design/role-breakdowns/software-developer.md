# Software Developer — Role Breakdown

## Core Responsibilities
- Implement features based on requirements and architecture
- Write clean, maintainable, and well-tested code
- Fix bugs and resolve production issues
- Write unit, integration, and end-to-end tests
- Refactor code for quality and performance
- Set up CI/CD pipelines and development tooling
- Perform code reviews and suggest improvements

## Action Points by Project Phase

### Implementation & Testing
- Build resume upload, parsing, and JD parsing modules
- Develop gap analysis, ATS scoring, and achievement rewriting logic
- Implement frontend/backend for pipeline workflow
- Write and maintain tests for all components
- Set up and maintain CI/CD pipelines

### Code Quality & Security
- Refactor code for maintainability and performance
- Ensure ATS-safe formatting and export (DOCX/PDF)
- Follow security best practices (input validation, no secrets in code)

### Continuous Improvement
- Address bugs and user feedback
- Optimize code and workflows based on analytics

## Triggers for Agent Activation
- New feature or bug fix required
- Test or CI/CD pipeline update
- Code review or refactoring task
- Security or performance improvement needed
