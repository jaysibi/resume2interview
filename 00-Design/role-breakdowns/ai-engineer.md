# AI Engineer — Role Breakdown

## Core Responsibilities
- Select and implement AI/ML models for resume and JD analysis
- Prepare, clean, and engineer data for training/inference
- Develop and maintain prompt templates for extraction, scoring, rewriting
- Integrate AI models into product workflows (APIs, microservices)
- Monitor, evaluate, and retrain models in production
- Ensure responsible AI practices (fairness, explainability, privacy)

## Action Points by Project Phase

### Data & Model Development
- Build and test resume parser and JD analyzer using Python libraries (pdfplumber, python-docx, PyMuPDF)
- Develop prompt templates for skill extraction, JD analysis, ATS match, achievement rewriting, summary optimization
- Implement and tune models for keyword/semantic matching and scoring
- Integrate OpenAI or similar LLMs for prompt-based tasks

### Integration & Monitoring
- Deploy models as APIs or embedded components
- Monitor model performance, drift, and user impact
- Document data sources, model choices, and evaluation metrics
- Ensure compliance with ethical and legal standards

### Continuous Improvement
- Update prompt templates and models based on feedback and analytics
- Collaborate with product and engineering teams for improvements

## Triggers for Agent Activation
- New prompt/model development or update
- Data pipeline or extraction task
- Model performance monitoring or retraining
- Responsible AI or compliance review
