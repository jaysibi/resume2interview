# Wireframes & Mockups — Resume Tailor Frontend

## Overview
This document outlines the wireframes and mockups for all key user flows in the Resume Tailor frontend. These designs serve as a blueprint for UI development and ensure a consistent, user-friendly experience.

## 1. Landing Page
- Welcome message and product value proposition
- Primary call-to-action buttons: "Upload Resume" and "Upload Job Description"
- Simple, clean layout with logo and navigation (if needed)

## 2. Resume Upload Screen
- File input for PDF/DOCX
- Drag-and-drop area
- Progress indicator for upload
- Error message for invalid file types
- Button to proceed after successful upload

## 3. JD Upload Screen
- File input for PDF/DOCX or text area for pasting JD
- Progress indicator for upload
- Error message for invalid file types or empty input
- Button to proceed after successful upload

## 4. Results Dashboard
- Display parsed resume and JD data (skills, experience, keywords)
- ATS match score and gap analysis
- Suggestions for improvement
- Download/export buttons for optimized resume and reports

## 5. Feedback & Export Screen
- User feedback form (rating, comments)
- Confirmation message after feedback submission
- Download links for resume and reports
